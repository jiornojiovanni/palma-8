- 00E0
- 00EE

- 4XNN
- 5XY0
- 6XNN
- 7XNN
- 8XY0
- 8XY1
- 8XY2
- 8XY3
- 8XY4
- 8XY5
- 8XY6

- 9XY0

- CXNN
- DXYN
- EX9E
- EXA1
- FX07
- FX0A
- FX1E
- FX29
- FX33
- FX55
- FX65

Build with `make` and run with `palmotto <chip8 rom>`

Install sdl2 to use this program.
