# palma-8 (Chip-8 C implementation with ncurses frontend)


Instructions currently implemented:
- 00E0
- 00EE
- 1NNN
- 2NNN
- 3XNN
- 4XNN
- 5XY0
- 6XNN
- 7XNN
- 8XY0
- 8XY1
- 8XY2
- 8XY3
- 8XY4
- 8XY5
- 8XY6
- 8XY7
- 8XYE
- 9XY0
- ANNN
- BNNN
- CXNN
- DXYN

Build with `make` and run with `palmotto <chip8 rom>`

Install ncurses to use this program.
